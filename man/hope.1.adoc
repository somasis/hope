= hope(1)
:source-highlighter: rouge

== Name

*hope* - a TAP-compliant, POSIX sh test harness

== Synopsis

*hope* _TEST_ ...

== Options

*-a*::
    Check if _any_ string matches, not all.

== Exit status

*0*::
    All tests passed.

*1*::
    A test failed.

== Examples

[source,shell]
----
if match -E "[0-9]{4}-[0-9]{2}-[0-9]{2}" "1998-05-28";then
    echo "that was a date!"
else
    echo "please give a date instead"
fi
----
